# Прогнозирование цен российских акций с MOEX

## Описание проекта

Проект загружает исторические данные российских акций с MOEX и строит прогноз цен с использованием линейной регрессии с лаговыми признаками, скользящими средними (MA5, MA10) и разностями цен.

Прогнозируемые значения OHLCV визуализируются в интерактивном интерфейсе на Streamlit.

Используемые акции по умолчанию:  
- Сбербанк (SBER)  
- Газпром (GAZP)  
- Лукойл (LKOH)  

Проект может использоваться для анализа трендов и демонстрации базовых подходов к прогнозированию временных рядов.

---

## Возможности

- Загрузка актуальных данных с MOEX
- Прогноз цены закрытия на несколько дней вперёд
- Визуализация исторических и прогнозных свечей (OHLCV)
- Настройка параметров:
  - горизонт прогноза
  - количество лагов
  - размер исторического окна
- Отображение расчетных финансовых и модельных показателей

## Запуск проекта

Вариант 1: через Docker

```bash
docker build -t moex_stock_forecast .
docker run -p 8501:8501 moex_stock_forecast
(После запуска приложение будет доступно по адресу:
http://localhost:8501)

Вариант 2: локально (без Docker)

```bash
pip install -r requirements.txt
```bash
streamlit run streamlit_app.py
```
## Структура проекта
- 'streamlit_app.py' — интерфейс приложения
- 'model.py' — модель прогнозирования
- 'moex_loader.py' — загрузка данных с MOEX
- 'api.py' — API (если используется)
- 'Dockerfile' — контейнеризация проекта
