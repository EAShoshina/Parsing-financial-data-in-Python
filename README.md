# Прогнозирование цен российских акций с MOEX

## Описание проекта

Данный проект загружает исторические данные российских акций с MOEX и строит прогноз цен с помощью линейной регрессии с лагами, скользящими средними (MA5, MA10) и разностями цен. Прогнозируемые свечи OHLCV визуализируются через Streamlit.

Используемые акции по умолчанию:  
- Сбербанк (SBER)  
- Газпром (GAZP)  
- Лукойл (LKOH)  

Проект подходит для анализа трендов и демонстрации прогнозирования на ближайшие дни.

---

## Возможности

- Загрузка последних данных с MOEX.  
- Прогнозирование цены закрытия на ближайшие дни.  
- Визуализация исторических и прогнозируемых свечей OHLCV.  
- Интерактивная настройка: количество дней прогноза, количество лагов, история для графика.  
- Формальная справка по финансовым и модельным показателям прямо под прогнозом.
